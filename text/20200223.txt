c++ 동적할당

1차원 배열 -> c에서는 malloc, free함수로 했지만,
c++에서는 new, delete 키워드로 함.

자료형 *변수명 = new 자료형;
delete 변수명;

자료형 *배열명 = new 자료형[크기];
delete[] 배열명;

2차원 배열 
ex) 행이 2, 열이 3인 2차원 배열을 동적할당할 경우
int **arr;
arr = new int* [2];
for(int n = 0; n< 2; n++)
	arr[n] = new int[3];

for(int n =0;n<2;n++)
	delete []arr[n];
delete []arr;

------------------------------------------------------------------------
class (클래스)
기본적으로 c++을 객체 지향 언어라 한다.
클래스란 클래스 내부 함수 및 변수를 통해 지속적으로 연산 및 반환을 하는 객체이다.

클래스 정의 형식
class 클래스명
{
	//맴버 변수(상태)
	//맴버 함수(행동)..
};

클래스는 맴버 변수와 맴버 함수로 이루어지며, 
클래스 정의 후 마지막에 세미콜론(;)을 붙여주어야 정의가 완료된다.

객체는 각각의 제조시설로 비유하며, 해당 제조시설(객체)를 활용하여 응용할 수 있는 툴이된다.

따라서 같은 객체에 다른 정보를 매개변수로 사용, 다른 결과값을 확인 할 수 있다.
------------------------------------------------------------------------
접근지정자 / 접근한정자
private:
class내에서 사용되어지는 맴버
public:
class외부에서도 사용될 수 있는 맴버
protected:
다른 class에서 가져갈 수 있는 맴버
